# Lyra dashboard — product voice

Single source of truth for UI copy in the dashboard.

## Rules

- **Nav and chrome:** Use **Title Case** for section names and navigation labels (e.g. Portfolio, Engine, Agents).
- **Primary actions and CTAs:** Use **sentence case** (e.g. Onboard project, sync now, Retry). Buttons and inline actions stay lowercase unless they start a sentence.

## Examples

| Context        | Correct           | Avoid            |
|----------------|-------------------|------------------|
| Sidebar nav    | Portfolio, Engine | portfolio, engine |
| Section header | Portfolio         | portfolio        |
| Button         | Onboard project   | Onboard Project  |
| Button         | sync now          | Sync Now         |
| Modal title    | Onboard project   | Onboard Project  |

When adding new screens or buttons, align with this so the app stays consistent.
